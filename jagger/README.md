# jagger

[jagger](https://www.tkl.iis.u-tokyo.ac.jp/~ynaga/jagger/) is a fast,
accurate, and space-efficient morphological analyzer from University of Tokyo.

## usage

See [Makefile](Makefile) for details.

```bash
make build
```

Jagger reads input from stdin or files.
